About This Site

This is a simple site with the intent to showcase common web design and development ideas currently in practise (2012).

NotAnotherWebsite is an in-browser rapid prototype.

Continuous deployment is used and highly effective for incremental improvements.

To build upon a successful framework such as Foundation 3 allows for greater productivity and stability for production websites.

Simplicity and design affordance remains the top priority for an excellent user experience.
