DOCTYPE Evolution:

HTML has come a long way since the early days of the web, but its' primary purpose remains to give structure to content. There are many new functionalities in HTML5 but cross-browser support is incomplete for now (2012).

HTML5: 

<!DOCTYPE HTML>

Boilerplates

The boilerplate is simply a template with successful code that aids with development to standards and ensure compatibility. As always, the community encourages developers to use and derive templates suitable for their project.

HTML5boilerplate is probably the most popular and successful of its' kind in use today.


CSS

The styling language of the web has grown leaps and bounds and remains instrumental in the separation of layout and content. Beautiful interfaces are now possible with pure CSS3 markup that enable drastically lower page load times in comparison with yesteryear's image laden interfaces. Although CSS3's guidelines are yet to be fully implemented across the major browsers, there continues to be ground-breaking advances for this community of front-end designers.


CSS Reset

CSS Reset is used to align the inconsistent behaviours of different browsers. The first person to write about this topic is Eric Meyer in 2001 on his website.


Grid System

Designers have known the grid for a long time. To take this design tool to the web is a recent phenomenon of the past several years. Most developers are probably familiar with Blueprint or 960gs. The usage of the grid allow for precise and orderly placement of elements on the web page. The most recent improvement to involve the grid is responsive design with CSS3 media queries.
This website which is built upon Foundation 3 also uses a grid system.


Design Affordances

Typical web users will spend less than 10 seconds looking at a web page to determine if it contains anything that interests them. If this first impression is confusing and unpleasant, the user will most likely leave. If the user cannot find the content they wish and yet, it resides on that website, they may also quickly leave. Therefore, it is of utmost importance in design to accomodate the user.
Affordance in design allows the user to be aware of the purpose of the design object. An object that is meant to be a button should be familiar to a user that it is pressable. A typical website may encompass a logo, navigation, content area, or search. If these components do not conform to most layouts, it may frustrate users.
Web designers make wise decisions with regards to the layout and graphical accents to a website because every first impression may be the last impression.


Rounded Box Corners

The obsession with rounded corners is well-documented in web design. From images to funny techniques, we have seen it all.
Now, we can easily accomplish them with CSS3.


Gradients

Another beautiful design element are gradients. When properly done by the designer, it can add depth and contrast to a website. Seemingly simplistic elements as such are part of colour theory and is tremendously important in a fabulous design. Believe it or not, properly placed lines and shadows allow for a minimalist and modern design that is both pleasing and easy to use.
Minimalism is a fine art.


Transitions

Transitions are previously the domain of JQuery but CSS3 is now part of the game. As with any other design element, they can be tremendously beautiful or tremendously annoying. This element rests with the expert decision of the designer to give a website just enough effects for a finely crafted user experience.
